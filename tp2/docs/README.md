# Docs TP2 SIA

Esta carpeta consolida la consigna, el marco teorico y el estado actual del repo para el TP2 de SIA sobre algoritmos geneticos.

## Fuentes principales

- `SIA - TP2 - 2026 1Q.pdf`: consigna oficial del TP.
- `Algoritmos Genéticos (2) 3.59.25 PM.pdf`: teoria vista en clase.

## Archivos de esta carpeta

- `consigna-tp2.md`: resumen operativo de la consigna del ejercicio 2.
- `marco-teorico-ag.md`: resumen corto de conceptos teoricos relevantes.
- `guia-diseno-y-defensa.md`: preguntas que conviene poder responder y criterios de diseno para el TP.
- `implementacion-actual-del-repo.md`: que soporta hoy el repo y donde hay diferencias entre consigna, README y punto de entrada principal.
- `mutacion-ag.md`: detalle de tipos de mutacion y su rol en diversidad.
- `supervivencia-ag.md`: detalle de supervivencia aditiva, exclusiva y brecha generacional.

## Orden sugerido de lectura

1. `consigna-tp2.md`
2. `marco-teorico-ag.md`
3. `guia-diseno-y-defensa.md`
4. `implementacion-actual-del-repo.md`
5. `mutacion-ag.md` y `supervivencia-ag.md`

## Nota de consistencia

- La fuente de verdad academica sigue siendo la consigna oficial y la teoria de clase.
- La fuente de verdad sobre el comportamiento actual del software es el codigo del repo.
- Si alguna parte del repo no coincide todavia con la cobertura pedida por la consigna, conviene explicitarlo en el README final o en la presentacion.
